# JSKD3UZD

1. Projekts ir uzdevumu pārvaldnieks (To-Do List), kur var pievienot, rediģēt, dzēst un mainīt uzdevumu statusu (done / not-done).

2. Darbība: dati tiek ielādēti no `tasks.json`, lietotājs veic darbības, un tās tiek saglabātas serverī ar PHP un `fetch()`.

3. Faili:

* `index.php` – galvenais fails
* `TaskController.php` – apstrādā darbības
* `TaskModel.php` – strādā ar datiem
* `taskView.php` – interfeiss
* `app.js` – loģika (JS)
* `style.css` – dizains

4. Palaišana:

```bash
php -S localhost:8080
```

Atvērt: http://localhost:8080
